# Быстрое решение проблемы с регистрацией

## Проблема
При нажатии "Зарегистрироваться" происходит бесконечная загрузка, и письмо не отправляется на почту.

## Быстрое решение

### Вариант 1: Использовать упрощенную регистрацию (без email)

1. **Замените импорт** в файле, где используется регистрация:
   ```dart
   // Вместо:
   import 'package:cruch/screens/register_screen/register_screen.dart';
   
   // Используйте:
   import 'package:cruch/screens/register_screen/register_screen_simple.dart';
   ```

2. **Замените виджет**:
   ```dart
   // Вместо:
   RegisterScreen()
   
   // Используйте:
   RegisterScreenSimple()
   ```

### Вариант 2: Настроить email (рекомендуется)

1. **Создайте Gmail аккаунт** и включите двухфакторную аутентификацию

2. **Создайте пароль приложения**:
   - Google аккаунт → Безопасность → Пароли приложений
   - Создайте пароль для "Почта"

3. **Обновите** `lib/config/email_config.dart`:
   ```dart
   static const String senderEmail = 'ваш-email@gmail.com';
   static const String senderPassword = 'ваш-пароль-приложения';
   ```

4. **Перезапустите приложение**

## Проверка

После применения любого варианта:
1. Запустите приложение
2. Попробуйте зарегистрироваться
3. Проверьте консоль на ошибки
4. Регистрация должна работать без зависания

## Логи

В консоли должны появиться сообщения:
- `AuthService: Начинаем регистрацию для...`
- `EmailService: Начинаем отправку кода на...` (если используете email)
- `AuthServiceSimple: Регистрация успешна` (если используете упрощенную версию)
